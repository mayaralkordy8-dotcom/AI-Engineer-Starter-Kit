
#  AI Engineer Starter Kit — Daily Quote Sentiment Analyzer

A beginner-friendly project that fetches a random motivational quote from an external API and performs Sentiment Analysis using a Hugging Face pre-trained Transformer model.

##  Key Features
* **REST API Integration:** Fetches real-time random quotes from the `ZenQuotes API` without authentication.
* **AI Pipeline:** Leverages Hugging Face's `transformers` library to classify the quote's sentiment as Positive or Negative.
* **Reproducible Design:** Fully documented Notebook and strict dependency management via `requirements.txt`.
